# ROMAN TensorRT Models 

Libraries for exporting models used in [ROMAN](https://github.com/mit-acl/roman) to ONNX/TensorRT, and helper frameworks for running them.

### Installation

```
git clone https://github.com/liqyn/roman-trt-models
cd roman-trt-models

pip install ONNX_TensorRT_Tools/
pip install FastSAM_TensorRT/
pip install YOLOv8_TensorRT/
pip install DINOv2_TensorRT/
```
